## ğŸ“ How to Add Tags

### In Admin Panel (Product Master)

1. **Navigate to Product Master**
2. **Create/Edit Product**
3. **Go to "SEO & Meta" Tab**
4. **Find "Tags" Field**
5. **Enter tags separated by commas**
   ```
   Best Seller, New Arrival, Trending
   ```
6. **Save Product**

### Tag Format
- **Comma-separated**: `Best Seller, Trending, New`
- **Case-insensitive**: System recognizes "best seller" and "Best Seller"
- **Multiple tags**: You can add multiple tags per product
- **No limit**: Add as many tags as needed

## âœ… Best Practices

### 1. **Be Strategic**
- Don't overuse tags
- 2-3 tags per product maximum
- Use tags that add real value

### 2. **Keep Updated**
- Remove "New Arrival" after 30 days
- Update "Trending" based on sales data
- Refresh "Best Seller" monthly

### 3. **Consistency**
- Use standard tag names
- Avoid variations (use "Best Seller" not "Best-Seller")
- Maintain consistent capitalization

### 4. **Combine with Discounts**
- Use "Sale" tag with actual price discounts
- Combine "Limited Edition" with low stock
- Pair "Trending" with high ratings
